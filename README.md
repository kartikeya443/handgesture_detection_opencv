Note: This model is trained and tested for 7 gestures at present. These are:
1. A, B, C (American Sign Language)
2. Hello (open hand with palm facing to the camera)
3. Love (hand with index finger and thumb crossed)
4. Yes (thumb's up)
5. No (thumb's down)
